# listening-time-prediction
An XGboost model to predict podcast listening time, using cross-validation techniques. A late submission for the Kaggle competition [playground-series-s5e4](https://www.kaggle.com/competitions/playground-series-s5e4). This was a great dataset to work on xgboost models and cross-validation techniques I have learnt and have been wanting to implement!\
All of the work was done locally and then uploaded to GitHub (I may have not created a repo for this earlier and decided to just work on it locally :P)
